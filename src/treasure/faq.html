1.怎么参加夺宝？

1、首先选择您想要的商品，然后选择您要参与的人次数（参与人次越多，中奖概率越大）进行参与。

2、参与成功后，您会获得相应数量的参与号码。当该商品参与人次数达到总需人次后，系统开始计算唯一的幸运号码。

3、开奖后，若您参与号码中有与该商品的幸运号码一致的号码，则您获得该商品。

2.有读夺宝是公正的吗？怎么计算幸运号码的？

公平、公正是有读夺宝赖以生存的根本。夺宝的幸运号码由公式计算得出，公式中的两个数字都是不受人为干扰的。

幸运号码= ( 数值A/数值B ) 取余数+1000001

数值A = 该商品最后一个抢宝号码分配完毕时间点前，夺宝平台全站最后50个参与时间相加求和

数值B = 商品所需次数

幸运号码计算规则说明

1. 每个商品都有总参与人次的最低要求，参与该商品抽奖的人次达到最低人次要求的5分钟后开奖；
2. 商品的最后一个号码分配完毕后，将公示该分配时间点前本站全部商品的最后50个参与时间；
3. 将这50个时间的数值进行求和，得出数值A（每个时间按时、分、秒、毫秒的顺序组合，如20:15:25.362则为201525362），若时间为20:13:14.002则数值为20131402），为了防止用户批量进行毫秒级下单，毫秒百位为0，自动舍去，数字将从9位，修改成8位进行求和计算；
4. 总需参与人次数为数值B；
5. 数值A除以数值B得到的余数 + 原始数 10000001，得到最终幸运号码，拥有该幸运号码者，直接获得该商品。
注：余数是指整数除法中被除数未被除尽部分，如10÷3=3……1,1就是余数。

3.什么是夺宝币?

夺宝币是有读夺宝的代币，用户每花1元即可获得1个夺宝币；1个夺宝币可以直接购买1个夺宝号码。夺宝币无法提现。

4.怎样查看是否成为幸运用户？如何领取幸运商品？

个人中心有您的夺宝记录，点击对应的记录，即可知道该期夺宝的获得者； 如果您成为幸运用户，会收到短信通知，收到通知后，请点击“夺宝主页面右上角+号>我的夺宝>中奖记录>点击领取”,按提示填写真实有效收货地址，以便我们为您派发获得的商品。

5.夺宝获得的商品什么时候发货?

获得夺宝商品后请尽快领取提交收货地址，超过7天未领取，视为自动放弃该商品（系统不会默认发送到默认地址，每次中奖都必须领取商品及确认收货地址），领取成功后，我们将在7个工作日内派发中奖商品，派发周期以实际快递公司为主，不在夺宝服务范围。

6.支付出现问题应该如何解决？

如果您出现在支付平台（如支付宝、微信支付、网银等）扣款成功但无法完成购买夺宝情况，那可能是银行单边账等原因。已扣款项将会在5-7个工作日原路退回你的支付账户； 如果您在支付平台扣款成功并生成夺宝失败订单，可能是因为支付完成时当期商品的夺宝号码已经分配完毕。该笔款项将以余额形式充值到有读账户。可在”个人中心>我的账户>账户明细“中查看。

7.收到的商品可以退换货吗？

所有商品一律不退货。非质量问题，不在三包范围内，不给予换货（收到商品5日内）；请尽量亲自签收并当面拆箱验货，如发现运输途中造成商品损坏，请不要签收，可以拒签退回。

8.夺宝获得的商品如何保修？

夺宝所有商品均从正规渠道采购，100%正品，可享受厂家所提供的全国联保服务。售后标准均按照该商品的售后标准进行。

9.我可以修改已确认的收货地址吗？

获得夺宝商品后一旦确认收货地址，无法二次修改。请您慎重填写！ 如因用户个人原因，导致快递无法正常派发，或者误发，用户承担所有责任以及二次运输费用。

10.可以挑选夺宝商品的颜色或型号吗？

用户获得的夺宝商品按参与夺宝时的商品名称来寄送，颜色型号均不能挑选或者更改。
